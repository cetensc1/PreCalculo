         \[ A = \{x \in \N | x < 5\}\]
         \[ B = \{x \in \Z | 3x + 4 = 13\}\]
         \[ C = \{x \in \R | x^2 + x – 12 = 0 \} \]
   é correto afirmar que: 
   \begin{enumerate}
   \item $A \cap B = \{ 1,2,3\}$
   \item $A \cup B = B$
   \item $A - B = \{0,1,2,4\}$
   \item $A \cup B \subset C$
   \item $B \cup C= \R$
  \end{enumerate}
